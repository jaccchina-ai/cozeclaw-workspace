## 🔍 初步框架

### Phase 2: T日晚间评分维度

#### 1. 涨停强度 (权重: 待定)

| 指标 | 说明 | 评分标准(待定) |
|------|------|---------------|
| 封板时间 | 首次涨停时间 | 9:30-10:00 得高分 |
| 封单金额 | 涨停时封单量 | 封单越大越好 |
| 开板次数 | 是否反复开板 | 0次最佳 |
| 最后封板时间 | 最终封板时间 | 越早越好 |

#### 2. 板块热度 (权重: 待定)

| 指标 | 说明 | 评分标准(待定) |
|------|------|---------------|
| 板块涨停数 | 板块内涨停股数量 | ≥3只得高分 |
| 板块涨幅 | 板块整体涨幅 | ≥2%得高分 |
| 板块地位 | 是否为当日主线 | 主线板块加分 |
| 板块持续性 | 连续几日活跃 | 持续性强加分 |

#### 3. 资金认可度 (权重: 待定)

| 指标 | 说明 | 评分标准(待定) |
|------|------|---------------|
| 主力净流入 | 主力资金净买入额 | 正值越大越好 |
| 游资参与 | 龙虎榜游资买入 | 知名游资加分 |
| 机构参与 | 机构专用席位买入 | 有机构买入加分 |
| 换手率 | 当日换手率 | 适中为佳(5%-15%) |

#### 4. 技术形态 (权重: 待定)

| 指标 | 说明 | 评分标准(待定) |
|------|------|---------------|
| 突破形态 | 是否突破关键位置 | 突破压力位加分 |
| 均线多头 | 均线排列形态 | 多头排列加分 |
| 量价配合 | 成交量放大情况 | 放量涨停加分 |
| 前期压力 | 距离前期高点 | 创新高最佳 |

#### 5. 基本面支撑 (权重: 待定)

| 指标 | 说明 | 评分标准(待定) |
|------|------|---------------|
| 业绩增长 | 最近财报表现 | 正增长加分 |
| 题材热度 | 当前热门题材 | 热门题材加分 |
| 逻辑清晰度 | 涨停理由合理性 | 逻辑清晰加分 |
| 市值大小 | 总市值规模 | 适中市值为佳 |
