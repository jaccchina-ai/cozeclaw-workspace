# Task T01: A股龙头选股策略

> **版本**: 1.0.0  
> **状态**: ✅ Active  
> **最后更新**: 2026-03-02

---

## 📌 任务概述

基于涨停板策略的A股龙头股选股系统，通过T日晚间和T+1日早盘两次筛选，精准捕捉最有望继续涨停的龙头股票。

---

## 🎯 核心目标

在交易日T日晚上8点对当日涨停股进行初步评分，筛选出10只最有潜力的观察标的；在T+1日早上9:25结合竞价数据进行最终筛选，推荐3只涨停概率最高的股票，并提供仓位建议和风险提示。

---

## 🔄 完整工作流程

### Phase 1: 交易日判断

**执行时间**: 任意时间  
**数据来源**: Tushare trade_cal 接口

```
输入: 日期
处理: 查询交易日历
输出: 是否为交易日 (boolean)
```

**API调用示例**:
```python
import tushare as ts

# 查询某日是否为交易日
pro = ts.pro_api('your_token')
cal = pro.trade_cal(exchange='SSE', start_date='20260302', end_date='20260302')
is_trading_day = cal.iloc[0]['is_open'] == '1'
```

---

### Phase 2: T日晚间初步筛选

**执行时间**: T日 20:00 (北京时间)  
**筛选目标**: 10只观察标的

#### 2.1 数据采集

| 数据类型 | API接口 | 字段说明 |
|---------|--------|---------|
| **涨停股列表** | `limit_list_d` | 当日涨停股票详细信息 |
| **涨停原因** | `limit_list_d` | 涨停原因分类 |
| **板块信息** | `concept_detail` | 所属概念板块 |
| **资金流向** | `moneyflow` | 主力资金进出情况 |
| **龙虎榜** | `top_list` | 游资机构买卖明细 |

#### 2.2 评分维度

待用户补充详细评分逻辑...

**初步评分框架**:
- 涨停强度 (封板时间、封单金额)
- 板块热度 (板块涨停数量、板块涨幅)
- 资金认可度 (主力资金净流入、游资参与)
- 技术形态 (突破形态、量价配合)
- 基本面支撑 (业绩、题材、逻辑)

### Phase 3: T+1日早盘最终推荐

**执行时间**: T+1日 09:25 (集合竞价结束)  
**推荐目标**: 3只最终标的

#### 3.1 竞价数据采集

| 数据类型 | API接口 | 字段说明 |
|---------|--------|---------|
| **竞价数据** | `stk_premarket_a` | 集合竞价价格、成交量 |
| **大盘竞价** | `index_premarket_a` | 指数竞价情况 |
| **板块竞价** | 板块成分股竞价聚合 | 板块整体竞价表现 |

#### 3.2 二次筛选逻辑

待用户补充详细筛选逻辑...

**筛选框架**:
- 竞价高开幅度 (理想区间: +2% ~ +5%)
- 竞价成交量放大倍数 (理想区间: 1.5x ~ 3x)
- 板块竞价一致性 (板块内多只股票高开)
- 竞价大单情况 (大单买入占比)
