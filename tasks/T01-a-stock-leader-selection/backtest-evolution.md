## 📊 回测框架

### 回测基础参数

| 参数 | 建议值 | 说明 |
|------|--------|------|
| **回测周期** | 近1年/近3年 | 覆盖不同市场环境 |
| **初始资金** | 100万 | 模拟本金 |
| **交易成本** | 0.15% | 佣金+印花税+滑点 |
| **基准指数** | 沪深300 / 创业板指 | 对比基准 |

### 回测评估指标

| 指标 | 说明 | 目标值 |
|------|------|--------|
| **总收益率** | 累计收益 | > 基准收益 |
| **年化收益率** | 年化收益 | > 20% |
| **最大回撤** | 最大亏损幅度 | < 20% |
| **夏普比率** | 风险调整收益 | > 1.5 |
| **胜率** | 盈利交易占比 | > 50% |
| **盈亏比** | 平均盈利/平均亏损 | > 1.5 |
| **交易次数** | 总交易次数 | 充足样本量 |
| **持仓周期** | 平均持仓天数 | 1-3天 |

### 回测场景分析

| 市场环境 | 时间段 | 预期表现 |
|---------|--------|---------|
| **牛市** | 指数持续上涨 | 高收益 |
| **震荡市** | 指数区间波动 | 稳定收益 |
| **熊市** | 指数持续下跌 | 控制回撤 |
| **急跌** | 短期大幅下跌 | 规避风险 |

## 🔄 策略进化方向

### 短期优化 (版本 1.1.0)

| 优化点 | 说明 | 预期收益 |
|--------|------|---------|
| 调整评分权重 | 根据回测优化权重分配 | 提升胜率 |
| 增加筛选条件 | 补充遗漏的重要指标 | 提升准确率 |
| 优化止损策略 | 动态止损而非固定止损 | 降低回撤 |

### 中期优化 (版本 1.2.0)

| 优化点 | 说明 | 预期收益 |

| 自适应参数 | 根据市场环境动态调整 | 适应不同行情 |

### 长期优化 (版本 2.0.0)

|--------|------|---------|

| 多市场扩展 | 港股/美股市场适配 | 扩展机会 |
| 实时风控系统 | 动态风险监控 | 降低极端风险 |

## 🔬 自适应机制

### 参数自适应

```
市场强势 → 激进参数
├─ 止损放宽: -5%
├─ 仓位提高: 80%
└─ 选股放宽: 评分>70

市场平稳 → 平衡参数
├─ 止损适中: -3%
├─ 仓位适中: 50%
└─ 选股适中: 评分>75

市场弱势 → 保守参数
├─ 止损收紧: -2%
├─ 仓位降低: 20%
└─ 选股严格: 评分>85
```

### 学习机制

```
每日复盘
├─ 记录所有信号
├─ 记录实际结果
└─ 分析偏差原因

每周总结
├─ 统计胜率盈亏比
├─ 识别问题模式
└─ 调整参数权重

每月优化
├─ 全面回测验证
├─ 更新评分模型
└─ 发布新版本
```
