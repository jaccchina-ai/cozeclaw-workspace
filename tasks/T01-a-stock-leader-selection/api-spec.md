```markdown
### 接口名称: [接口名称]

**用途**: [接口用途说明]

**请求方式**: GET / POST

**端点**: `/api/v1/endpoint`

**认证方式**: 
- Header: `Authorization: Bearer {token}`

**请求参数**:
| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| param1 | string | 是 | 参数说明 |

**请求示例**:
\`\`\`bash
curl -X GET "https://api.example.com/api/v1/endpoint?param1=value1" \
  -H "Authorization: Bearer YOUR_TOKEN"
\`\`\`

**响应示例**:
\`\`\`json
{
  "code": 0,
  "message": "success",
  "data": {
    // 返回数据结构
  }
}
\`\`\`
```
